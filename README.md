## study_git

- [progit](http://git-scm.com/book/en/v2)을 읽으며 공부한 내용입니다.

### 목적

- 지금까지 git을 통해 프로젝트를 관리했지만 git add, commit, rest, push, pull 이외에는 사용해본적이 없었다.
- 그래서 다양한 기능을 가지고 있는 git을 잘 활용해보고 싶다는 생각에 progit2를 통해 git에 대해 정리하고 그 내용을 올리려고 합니다.
